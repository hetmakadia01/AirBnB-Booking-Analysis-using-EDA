# 🏡 Airbnb Booking Analysis using EDA

## 🚀 Overview
This project focuses on performing Exploratory Data Analysis (EDA) on Airbnb booking data to uncover patterns, trends, and insights. It helps understand pricing, room types, locations, and customer behavior in the Airbnb marketplace.

---

## 🎯 Features
- 📊 Data cleaning and preprocessing  
- 📈 Exploratory Data Analysis (EDA)  
- 🏠 Room type distribution analysis  
- 💰 Price distribution and comparison  
- 🌍 Location-based insights  
- 📉 Time-series analysis of reviews  

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

```bash
pip install pandas numpy matplotlib seaborn
```

## ⚙️ How It Works
1. Load Airbnb dataset  
2. Handle missing values and clean data  
3. Convert data types (price, dates, etc.)  
4. Remove duplicates  
5. Perform data visualization  
6. Analyze patterns and insights  

---

## 📊 Key Analysis
- Distribution of listing prices  
- Room type comparison (Entire home, Private room, etc.)  
- Neighborhood-wise listing distribution  
- Price vs room type relationship  
- Reviews trend over time
